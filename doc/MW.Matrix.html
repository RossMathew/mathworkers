Class: Matrix

Matrix

new Matrix(nrowsopt, ncolsopt)

Matrix class. A wrapper around an array of Float64Arrays (Array.) with several matrix operations defined, including worker parallelized operations.
Parameters:
Name Type Attributes Description
nrows number <optional>
 the number of rows in the matrix. If not provided or less than 1, a Matrix object is still created but with a null array.
ncols number <optional>
 the number of columns in the matrix. If not provided or less than 1, a Matrix object is still created but with a null array.
